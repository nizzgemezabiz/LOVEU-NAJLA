# 🌸 Pinterest Aesthetic Love Scrapbook for Najla

Sebuah proyek website scrapbook digital premium kelas dunia yang dirancang khusus sebagai hadiah romantis bernuansa elegan, minimalis, dan mewah untuk **Najla**. Dibuat menggunakan standar desain Apple Human Interface dan tata letak galeri Pinterest berbasis Glassmorphic, website ini sepenuhnya siap untuk di-deploy ke **GitHub Pages**.
